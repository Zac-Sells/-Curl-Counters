User Stories

Jill Scenario #1: Jill's Yoga Journey

As a beginner, I need to be able to find information on different types of workouts so I have an idea of where to start.

As a beginner, I need to see a list of gyms that are in my area.

As a beginner, I need to see workouts that are labeled as easy difficulty.

As someone with health concerns, I need to find low-impact exercises.

As a yoga student, I want to be able to track the poses I have learned.

As a yoga student, I need to be able to see information on yoga poses so I can practice my technique at home.

As a beginner, I need to be able to input that I attended a yoga class and how long the class lasted.

As a beginner, I want to see a chart of how often I have attended yoga classes so I can stay motivated.

Jill Scenario #2: Jill's Evolution

As a beginner, I want to be able to easily find and sign up for local walking groups so that I can stay active and meet new people.

As a beginner, I want to access the walking group's schedule and location so that I can plan my transportation accordingly.

As a beginner, I want to receive guidance and encouragement from a certified personal trainer to improve my fitness.

As a beginner, I need to be able to set new fitness goals for specific workouts, including walking, swimming, and yoga.

As a beginner, I need to be able to mark off when I hit a goal so that I can stay motivated.

As a beginner, I want to be able to easily access and review previous workout routines so that I can measure my progress.

As a beginner, I want to be able to receive reminders for a scheduled workout so I don't lose motivation.

As a beginner, I want to be able to share posts of my progress with my family and friends so that I can get more support.

As a beginner, I want to be able to search for training plans based on difficulty level.

As a swimmer, I want to be able to enter my time for various distances.
