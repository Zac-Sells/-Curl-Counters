Geolocation

Description

As a beginner, I need to see a list of gyms in my area.

As a beginner, I want to be able to easily find and sign up for local walking groups so that I can stay active and meet new people.

As a beginner, I want to access the walking group's schedule and location to plan my transportation accordingly.

As a beginner, I need to be able to see which gyms have indoor pools.

As a weightlifter, I want to be able to search for gyms in my area.

As a weightlifter, I would like to be able to coordinate with other people when they are heading to the gym.

As a weightlifter, I would like to see nearby gyms and addresses.

Constraints

The app will need to handle many users and gyms, which may require significant resources and infrastructure.

The app will need access to information about gyms, such as location, amenities, and user reviews. This data may be challenging to acquire, particularly if gyms do not have a solid online presence.

Comments

Based on Jill and Matt’s scenarios, constant use of GPS can drain the battery of the user's device; the feature should be designed to minimize the impact on battery life.
