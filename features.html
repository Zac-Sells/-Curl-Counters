Feature List

Exercise tracking

Description

As a yoga student, I want to be able to track the poses I have learned. As a beginner, I need to be able to input that I attended a yoga class and how long the class lasted. As a beginner, I need to be able to input that I attended a yoga class and how long the class lasted. As a beginner, I want to be able to easily access and review previous workout routines so that I can measure my progress. As a beginner, I want to see a chart of how often I have attended yoga classes so I can stay motivated As a beginner, I want to be able to schedule my workout routine in advance so that I can stay organized and on track. As a swimmer, I want to be able to enter my time for various distances. As a weightlifter, I want to track my workout routines. As a bicyclist, I want to track my fitness level. As a bicyclist, I want to track my progress. As a bicyclist, I would like to record my heartbeat. As a weightlifter, I would like to see visual representation of my progress.

Constraints

Wide variety of exercises and equipment to keep track of. May be hard to include everything in the database. Organization may be difficult.

Comments

Based on Jill, Matt and Chris’ scenarios

Weight tracking

Description

As a weightlifter, I need to be able to track my calorie intake. As a weightlifter, I need to be able to calculate how many calories I burn during my workout. As a weightlifter, I want reminders telling me to eat more when I miss my caloric goal. As a weightlifter, I want ways to find my body fat percentage. As a weightlifter, I want to track my weight gain/loss. As a weightlifter, I would like to track my caloric intake/burned. As a weightlifter, I would like to see visual representation of my progress

Constraints

Will need to find different data on how many calories you burn doing each different exercise. Calculating accurate body fat percentage is difficult to do without certain equipment.

Comments

Based on Matt and Chris’ scenario

Login account

Description

As a beginner, I want to be able to easily find and sign up for local walking groups so that I can stay active and meet new people. As a weightlifter, I want to be able to control what my friends see on my results. As a weightlifter, I need to be able to block people if they are harassing me. As a weightlifter, I would like to be able to coordinate with other people when they are heading to the gym. As a weightlifter, I need my data to be secured from nefarious users. As a weightlifter, I want to be able to compare my stats to the entire user base. As a weightlifter, I want to be able to create challenges in my friend group.

Constraints

The user has to create their account and restrict their privacy settings on however they want.

Based on Matt’s scenario
